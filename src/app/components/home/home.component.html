Welcome to Horde Model Reference

The authoritative source for AI model metadata in the AI-Horde ecosystem. Explore, search, and manage model references for distributed AI generation.

What is AI-Horde?

AI-Horde is a free, community-run, distributed compute network for AI generation tasks. AI-Horde lets volunteers share their computer power to help others create AI art and text for free.

Volunteers run workers (GPU nodes) that process generation requests. Users submit requests for images, text, or other AI tasks. The system uses kudos (priority points) to fairly distribute resources, but anonymous users can still use the service for free at lower priority.

This Model Reference database ensures all workers and clients have accurate, verified metadata about which models exist, where to download them, and how to use them. It's the single source of truth that keeps the distributed system working reliably.

Key Concepts

Baseline

The foundation model architecture (SD1.5, SDXL, Flux). Determines GPU requirements and generation speed. Hover over the term anywhere in the app for details.

Parameters

Model size in billions (for text models). Larger = better quality but slower generation and higher GPU requirements.

Active Workers

Number of GPUs currently serving a model. More workers = faster queue times. Zero workers = model unavailable.

Kudos

Priority currency earned by running workers. Higher kudos = faster generation. Never expires, cannot be bought.
